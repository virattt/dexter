# 株主還元分析スキル

## ワークフローチェックリスト

```
株主還元分析:
- [ ] Step 1: 企業検索
- [ ] Step 2: 財務時系列データ取得（5年分）
- [ ] Step 3: 決算短信で最新の配当予想を確認
- [ ] Step 4: 配当利回りランキングで相対位置を確認
- [ ] Step 5: 株主還元指標の分析
- [ ] Step 6: レポート出力
```

## Step 1: 企業検索

`jp_search_company` で edinet_code を取得。

## Step 2: 財務データ取得

`jp_financials` で通期データ5年分を取得。以下を抽出:
- 1株当たり配当金（DPS）
- 配当性向（payout_ratio）
- 配当利回り（dividend_yield）
- EPS
- 自己株取得額（取得可能な場合）

## Step 3: 最新配当予想

`jp_earnings` で直近の決算短信から配当予想を確認。

## Step 4: ランキング確認

`jp_ranking` で dividend-yield と payout-ratio のランキングを取得。

## Step 5: 分析

### 5.1 配当推移
- 過去5年の DPS 推移
- 増配/減配/維持のパターン
- 連続増配年数

### 5.2 配当性向
- 過去5年の配当性向推移
- 利益変動に対する配当の安定性
- 30-50%が健全な目安

### 5.3 配当利回り
- 現在の配当利回り
- 過去3年の配当利回りレンジ
- 業種内での位置

### 5.4 総還元性向
- (配当 + 自己株取得) ÷ 純利益
- 東証の資本効率改善要請への対応状況

## Step 6: レポート出力

1. **配当サマリー**: 現在の配当利回り、DPS、配当性向
2. **配当推移表**: 過去5年のDPS・配当性向・利回り
3. **業種内ポジション**: 配当利回りランキング
4. **総還元性向**: 配当+自己株買いの総合評価
5. **今後の見通し**: 増配余地、利益成長との関係
6. **投資判断のポイント**: インカムゲイン視点での評価
7. **データ出典**: EDINET DB
