# バリュー株スクリーニングスキル（清原式）

## 概要

清原達郎氏の投資手法に基づくネットキャッシュ割安株スクリーニング。
「時価総額よりもネットキャッシュ（現金+有価証券-有利子負債）が多い企業」を見つける。

## ワークフローチェックリスト

```
バリュー株スクリーニング:
- [ ] Step 1: PBRランキングで低PBR銘柄を抽出
- [ ] Step 2: ROEランキングで収益性を確認
- [ ] Step 3: スクリーニング条件でフィルタ
- [ ] Step 4: 上位銘柄の詳細分析
- [ ] Step 5: レポート出力
```

## Step 1: 低PBR銘柄の抽出

`jp_ranking` で PER ランキング（低い順）を取得。
※ EDINET DBのランキングAPIで直接PBR順を取得できない場合は `jp_screening` を使う。

## Step 2: 収益性の確認

`jp_ranking` で ROE ランキングを取得し、クロスリファレンス。

## Step 3: スクリーニング

`jp_screening` で以下の条件を適用:

### 基本条件（清原式）
- PBR < 1.0（解散価値割れ）
- ROE >= 8%（収益性は健全）
- equity_ratio >= 40%（財務体質が健全）

### 追加条件（オプション）
- 配当利回り >= 2.5%（インカムゲインも確保）
- 営業利益率 >= 5%（本業の収益力）

## Step 4: 上位銘柄の詳細分析

スクリーニング結果の上位5-10銘柄について:
1. `jp_company_info` で企業概要を取得
2. `jp_financials` で3年分の財務推移を確認
3. `jp_analysis` でヘルススコアを確認

### ネットキャッシュ比率の推定
財務データから以下を計算:
- ネットキャッシュ = 現金同等物 + 投資有価証券 - 有利子負債
- ネットキャッシュ比率 = ネットキャッシュ ÷ 時価総額
- 1.0以上なら「タダで買える企業」

## Step 5: レポート出力

1. **スクリーニング条件**: 使用した条件の一覧
2. **結果サマリー**: ヒット件数と上位銘柄リスト（表形式）
3. **個別銘柄分析**: 上位5銘柄の詳細
   - 企業概要（業種、時価総額）
   - バリュエーション（PBR、PER、配当利回り）
   - 財務健全性（自己資本比率、ヘルススコア）
   - ネットキャッシュ比率（推定）
4. **注意事項**:
   - 低PBRには理由がある場合がある（構造的問題、成長性の欠如）
   - ネットキャッシュ比率は有報の数値に基づく推定値
   - 投資判断は自己責任で
5. **データ出典**: EDINET DB
